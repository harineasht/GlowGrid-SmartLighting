# 🌟 GlowGrid-AI

### AI-Based Smart Street Light Monitoring & Fault Detection System

---

## 🚀 Overview

**GlowGrid-AI** is an intelligent street lighting solution designed to transform traditional urban infrastructure into a **smart, efficient, and responsive lighting network**.

By combining **AI, IoT, and real-time monitoring**, this system detects faults, monitors power usage, and ensures street lights operate efficiently — reducing energy waste and improving public safety.

---

## 💡 Key Features

✨ **Real-Time Monitoring**

* Continuously monitors street light status (ON/OFF)
* Tracks voltage and current using sensors

⚡ **Fault Detection System**

* Detects failures like light outages or abnormal power consumption
* Instantly identifies faulty street lights

📡 **Wireless Communication (LoRa)**

* Long-range, low-power communication between nodes
* Reliable data transmission for large-scale deployments

🧠 **AI-Based Insights (Planned/Optional)**

* Predictive maintenance
* Usage pattern analysis
* Smart energy optimization

🌍 **Smart City Integration**

* Scalable for urban/rural infrastructure
* Supports centralized monitoring dashboards

---

## 🛠️ Hardware Components

* 🔹 ESP32 (Main Controller)
* 🔹 ACS712 (Current Sensor)
* 🔹 ZMPT101B (Voltage Sensor)
* 🔹 LoRa E220 Module
* 🔹 Street Light / Load (Bulb)

---

## 💻 Software & Technologies

* Arduino IDE
* Embedded C / C++
* IoT Architecture
* (Optional) AI/ML Models for analytics

---

## 🧩 System Architecture

1. Sensors measure **voltage & current** from street lights
2. ESP32 processes the data
3. Fault conditions are detected
4. Data is transmitted via **LoRa**
5. Central system/dashboard monitors all street lights
